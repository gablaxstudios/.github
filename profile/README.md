## Qui sommes-nous ?

Nous sommes **G&L Studios**, un duo de développeurs indépendants passionnés par le développement sur le jeu cubique **Minecraft**. Nous concevons **des plugins de tous types pour tous les goûts et tous les projets !**

Nous sommes membres de l'équipe technique de [**@DeryCube**](https://www.derycube.com) - Un serveur Minecraft FFA & Mini-Jeux où nous sommes tous les deux impliqués dans certains Mini-jeux.
